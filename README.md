# MOSSE
A modular audio system for making and consuming audio content on the go.
The device functions like an iPod, with a capactive touch clickwheel, as well as a capactive touch slider for tweaking parameters.
Music files will be able to be uploaded over the usb-c port.
I made this project because I wanted something to take with me on the go that was highgly capable of all things audio. I want to be able to record, produce, and play back any audio that I can imagine, and this is the first step on that journey.
